- `text-decoration:none`：去除`<a>`标签默认的下划线

# 6. 图像标签（行内元素）

`<img />` 标签用于定义 HTML 页面中的图像

- `src`属性：图像路径，==必须指定==
- `alt`属性：替换文本，图片无法显示时显示此文本
- `title`属性：提示文本，鼠标放在图像时显示此文本
- `width`属性：设置图像宽度
- `height`属性：设置图像高度

```HTML
<img src="图像URL" alt="替换文本" title="提示文本" />
```

## 常用图像格式

JPG PNG GIF WEBP AVIF

## 经验

一般宽度和高度设置其中一个就可以，另一个会等比缩放
现在宽高用 CSS 来设置比较好

# 9. 超链接标签（行内元素）

`<a>...</a>`，可以跳转项目内其他 HTML 文件或者外部链接

1. `href`属性：指定链接目标的 URL 地址
   - 空连接：`href="#"`
   - 下载链接：如果为压缩包格式文件（`.exe`/ `.zip`等等），点击则会下载文件
2. `target`属性：指定链接页面的打开方式，
   - `_self` 默认值（在当前窗口打开）
   - `_blank` 在新窗口打开
3. `title`属性：鼠标悬停显示文本

```HTML
<a href="URL" target="目标窗口弹出方式">...</a>
```

## 锚点链接

快速跳转到该页面中的某个位置。在 `href` 属性中，设置属性值为 `#名字`，找到目标位置标签，添加 `id` 属性，属性值跟上面名字相同即可

```HTML
<a href="#two">第二集</a>
...
<h3 id="two">第二集内容</h3>
```

跳转是直接硬切换，平滑切换可用下面 CSS 代码

```CSS
html {
	scroll-behavior:smooth;
}
```

# 10. 网页结构标签（块级元素）

## 有语义标签

可以看成是`<div>`标签的语义版

- `<header>`：网页页眉（头部）
- `<main>`：网页内容（每个网页只能有一个）
- `<nav>`：导航栏
- `<article>`：文章相关
- `<section>`：分块
- `<aside>`：侧边栏
- `<footer>`：网页页脚（底部）

## 无语义标签

- `<div>`：块级元素，用于布局结构，作为其他元素的容器
- `<span>`：行内元素，用于操作文本或其他行内元素的局部样式

# 11. 列表标签

## 11.1 无序列表

`<ul> </ul>` 定义无序列表，其内部用 `<li> </li>` 定义列表项

1. `<ul>`只能嵌套 `<li>`，`<li>`可以嵌套其他标签
2. 无序列表默认前面带有小圆点，可用 CSS 来解决：`li {list-style: none;}`

```HTML
<ul>
	<li>第一项</li>
	<li>第二项</li>
	<li>第三项</li>
</ul>
```

## 11.2 有序列表

`<ol> </ol>` 定义有序列表，其内部用 `<li> </li>` 定义列表项

> `<ol>`只能嵌套 `<li>`，`<li>`可以嵌套其他标签

```HTML
<ol>
	<li>第一项</li>
	<li>第二项</li>
	<li>第三项</li>
</ol>
```

### 经验

有序列表在开发中不常用，了解即可

## 11.3 描述列表

描述列表常用于对术语或名词进行解释和描述，定义列表的列表项前没有项目符号
`<dl> </dl>` 定义描述列表，其内部用 `<dt> </dt>` 定义描述对象，用 `<dd> </dd>` 来解释、修饰这个对象

> `<dl>`只能嵌套 `<dt>`、`<dd>`

```HTML
<dl>
	<dt>好吃的水果</dt>
	<dd>苹果</dd>
	<dd>香蕉</dd>
</dl>
```

# 12. 表格标签

## 12.1 基本组成

- `<table> </table>` 定义表格
- `<tr> </tr>` 定义表格的行
- `<th> </th>` 表头单元格，会加粗、居中显示
- `<td> </td>` 定义单元格数据

```HTML
<table>
	<tr>
		<td>表格文字</td>
	</tr>
</table>
```

## 12.2 表格结构标签

- `<thead> </thead>` 表格头部
- `<tbody> </tbody>` 表格主体
- `<tfoot> </tfoot>` 表格底部

```HTML
<table>
	<thead>
		<tr>
			<th>line1</th>
			<th>line2</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>content1</td>
			<td>content2</td>
		<tr>
	</tbody>
</table>
```

## 12.3 合并单元格（`<td>`属性）（少用）

- 跨行合并：`rowspan="合并单元格的个数"`
  > 最上侧单元格为目标单元格，写合并代码
- 跨列合并：`colspan="合并单元格的个数"`
  > 最左侧单元格为目标单元格，写合并代码

```HTML
<!DOCTYPE html>
<html lang="en">
<head>
...
</head>
<body>
    <table>
        <thead>
            <tr>
                <td>col1</td>
                <td>col2</td>
                <td>col3</td>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td colspan="2"></td>
                <td></td>
            </tr>
            <tr>
                <td rowspan="2"></td>
                <td></td>
                <td></td>
            </tr>
            <tr>
                <td></td>
                <td></td>
            </tr>
        </tbody>
    </table>
</body>
</html>
```

## 经验

在实际开发中用 CSS 去控制表格的各种属性，也很少涉及需要合并单元格的地方，合并单元格会使得表格难以维护，会影响响应式布局之类的（可以借助 AI 来写合并单元格的代码）

> [!表格边框]
> 单元格默认边框不是合并起来的，可以利用以下 CSS 代码进行合并
> `border-collapse:collapse;`

# 13. 表单标签

表单的核心部件有三部分：表单容器、表单控件、辅助标签

- 表单容器：`<form>`（定义表单容器，包裹表单控件）
- 表单控件：`<input>`、`<textarea>`、`<select>`、`<button>`等等
- 辅助标签：`<label>`（关联输入控件的文本标签等等）

## 3.1 表单容器

`<form> </form>` 定义表单容器

| 属性     | 属性值       | 作用                                          |
| -------- | ------------ | --------------------------------------------- |
| `action` | URL 地址     | 指定接收并处理表单数据的服务器程序的 URL 地址 |
| `method` | `GET`/`POST` | 设置表单数据的提交方式                        |
| `name`   | 名称         | 指定表单名称，区分同一个页面中多个表单容器    |

## 3.2 表单控件

### 3.2.1 `<input />`

输入单行文本

1.  `type`属性：
    - `text`文本框，默认宽度为 20 个字符
    - `password`密码字段，显示圆点
    - `checkbox`复选框
      - 利用`name`属性归组，`checked`属性添加默认选中状态
    - `radio`单选按钮
      - 利用`name`属性归组，`checked`属性添加默认选中状态
    - `submit`提交，将表单数据发送给服务器
    - `reset`重置，清除表单数据
    - `file`文件域，上传文件
      - 利用`multiple`属性允许多选文件，`accept`规定选择的文件类型（逗号分隔）
2.  `name` 属性：表单名称
3.  `value` 属性：规定该`<input/>`元素的值
4.  `placeholder`属性：文本框提示信息
5.  `required`属性：文本不能为空
6.  `autocomplete`属性：值为`off/on`，记录历史信息，默认打开

### 3.2.2 `<textarea></textarea>`

输入多行文本

1. `name`属性：表单名字
2. `placeholder`属性：提示信息
3. `cols`属性：文本列数，默认 20
4. `rows`属性：文本行数，默认 2

### 3.2.3 `<select> </select>` （少用）

定义下拉列表，`<option></option>`为列表项，但由于其默认样式不大美观，现实开发中很少用该标签，大多数都是自己写`<div>`来实现

> `<select>`标签中至少包含一对 `<option>`
> 在 `<option>`中定义 `select="selected"`，当前项即为默认选中项

```HTML
<form>
	<select>
		<option>选择1</option>
		<option select="selected">选择2</option>
		<option>选择3</option>
	</select>
</form>
```

### 3.2.4 `<button> </button>`

定义一个按钮，内部可写文字或者放图像

- `disabled`属性可禁用按钮

## 3.3 `<label>` 辅助标签

`<label>...</label>`标签为 `<input/>` 元素定义标注
`<label>`标签的 `for`属性要与相关元素的 `id`属性相同

> 可以理解为：如果只点击按钮来实现选定效果的话，面积太小了，用此方法让旁边的解释文本也实现同样效果，增加可点击面积

```HTML
<!-- 实现形式1 -->
<label for="sex">男<label/>
<input type="radio" name="sex" value="male"/>

<!-- 实现形式2 -->
<label for="sex"><input type="radio" name="sex" value="male" />男<label/>
```
