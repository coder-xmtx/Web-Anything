# 1. 后代选择器（常用）
可以选择父元素里面的子元素，给**所有**该子元素修改样式。
语法：`元素1 元素2 {}`，后代选择器可以是任意基础选择器的组合
> [!example]
> 想只给 `<ol>` 里面的 `<li>` 修改样式，不改变 `<ul>` 里面的 `<li>`
```HTML
<body>
    <ol>
        <li>hello</li>
        <li>hello</li>
    </ol>
    <ul>
        <li>ok</li>
        <li>ok</li>
    </ul>
 </body>
```
> 只需要这样编写：
```CSS
ol li {
	color: red;
}
```
> 类选择器和标签选择器也能组合
```CSS
.demo li a {
	color: red;
}
```

# 2. 子代选择器
只选择某元素最近的一级子元素
语法：`元素1 > 元素2 {}`
```CSS
div > p {
	color: red;
}
```

---

# 3. 并集选择器（常用）
可以选择多组选择器，同时为它们定义相同的样式
语法：`元素1,元素2 {}`
```CSS
div,
p {
	color: red;
}
```

# 4. 兄弟选择器
## 4.1 邻接兄弟选择器
选中某标签后面的第一个元素
```CSS
/*选择h2标签后面第一个平行的p标签*/
h2 + p {
	color:red;
}
```

## 4.2 通用兄弟选择器
选中某标签后面所有的某个元素
```CSS
/*选择h2标签后面所有平行的p标签*/
h2 ~ p {
	color:red;
}
```

# 5. 属性选择器
属性选择器可以根据元素特定属性来选择元素，这样可以不用借助类选择器或者id选择器

| 选择符             | 说明                      |
| --------------- | ----------------------- |
| `E[att]`        | 选择具有att属性的E元素           |
| `E[att="val"]`  | 选择具有att属性且属性值＝val的E元素   |
| `E[att^="val"]` | 选择具有att属性且以val开头的E元素    |
| `E[att$="val"]` | 选择具有att属性且以val结尾的E元素    |
| `E[att*="val"]` | 选择具有att属性且属性值中含有val的E元素 |

# 6. 状态伪类选择器

## 6.1 `:hover` 
选择鼠标指针位于其上的元素

```CSS
/*鼠标放置其上时变色*/
a:hover {
	color: skyblue;
}
```

### 经验
链接伪类选择器还有另外几个，不过用的最多的还是`:hover`，其他的基本没用过
`:hover` 也能用在其他标签上，不只是链接

## 6.2 `focus`
用于获取获得焦点的表单元素，比如让正在选取的输入框改变背景颜色
```CSS
input:focus {
	background-color: blue;
}
```

# 7. 结构伪类选择器
结构伪类选择根据文档结构来选择元素，常用根据父级选择器里面的子元素

- `E:first-child` 匹配父元素中第一个子元素E
- `E:last-child` 匹配父元素中最后一个子元素E
- `E:nth-child(n)`匹配父元素中第n个子元素E
	- n可以是数字（从1开始）
	- 关键字(`even`偶数，`odd`奇数)
	- 可以是以n为变量的公式（由0递增）

# 8. 表单伪类选择器
可以给表单的`disabled`/`checked`等属性设置样式
```CSS
button:disabled {
	opacity: 0.4;
}

/*注意：checked无法直接修改复选框的颜色*/
input:checked + label {
	color:blue
}
```

# 9. 伪元素选择器

| 选择符          | 简介                                 |
| --------------- | ------------------------------------ |
| `::before`      | 在元素内插入伪元素，作为第一个元素   |
| `::after`       | 在元素内插入伪元素，作为最后一个元素 | 
| `::placeholder` | 选择input或者textarea占位符          |

- `before` 和 `after` 创建一个元素，但是属于行内元素
- `before` 和 `after` 必须有 `content` 属性，没内容可写空引号
- `before` 在父元素内容的前面创建元素，`after` 在父元素内容的后面插入元素
